# 🎬 CineSphere

A full-stack Cinema Multiplex Management System built with PostgreSQL, Node.js, Express, HTML, CSS, JavaScript, Bootstrap, and Chart.js.

The application provides CRUD operations for cinema management, interactive analytics dashboards, and a RESTful API backed by PostgreSQL.

---

## 🚀 Features

- Complete cinema management system
- RESTful API with CRUD operations
- PostgreSQL database with relational schema
- Interactive analytics dashboard
- Revenue, bookings, and occupancy reports
- Dynamic charts using Chart.js
- Responsive Bootstrap interface

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL (Neon)
- **Charts:** Chart.js

## 🗄️ Database

The system is built on a normalized PostgreSQL database containing **20 relational tables** with:

- Primary Keys
- Foreign Keys
- CHECK Constraints
- UNIQUE Constraints
- NOT NULL Constraints
